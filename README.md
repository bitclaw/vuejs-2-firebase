# vuejs-2-firebase
VueJS 2 Firebase App

Firebase schema example:

```json
{
  "books":{
    "book1":{
        "author":"John",
        "title":"John Vincent",
        "url":"www.google.com"
    },
    "book2":{
        "author":"Wayne",
        "title":"Wayne's World",
        "url":"www.waynesworld.com"
    }
  }
 }
```

Initialize Vuejs2 project:

```shell
npm init
npm install vue-cli --save
vue init webpack vuejs-2-firebase

````
